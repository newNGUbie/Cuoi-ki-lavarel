# Dự Án E-commerce: Shop Đồ Gia Dụng Smart

Dự án website bán hàng đồ gia dụng được xây dựng trên nền tảng Laravel.

## Tính năng chính
- Quản lý sản phẩm, loại sản phẩm (Đồ gia dụng).
- Giỏ hàng và đặt hàng.
- Áp dụng mã giảm giá (Coupon).
- Tính phí vận chuyển theo khu vực.
- Theo dõi đơn hàng (Lịch sử mua hàng).
- Gửi email xác nhận đơn hàng và phản hồi liên hệ.
- Quản trị viên (Admin): Quản lý sản phẩm, đơn hàng, khách hàng và phản hồi liên hệ.

## Cài đặt
1. Clone repository.
2. Chạy `composer install`.
3. Cấu hình file `.env` (Database, Mailtrap).
4. Chạy `php artisan migrate --seed`.
5. Chạy `php artisan serve`.

## Thông tin sinh viên
- Môn học: Phát triển ứng dụng Web với Laravel.
- Đề tài: Xây dựng website bán đồ gia dụng.
